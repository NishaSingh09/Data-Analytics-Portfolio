# Bank Loan Analytics Dashboard

## Overview

This Power BI project analyzes loan portfolio performance and borrower behavior using a banking loan dataset.

The dashboard helps track:

* Total Loan Applications
* Total Funded Amount
* Total Amount Received
* Average Interest Rate
* Average Debt-to-Income Ratio (DTI)
* Good Loan vs Bad Loan Analysis
* Loan Purpose Analysis
* Loan Status Distribution
* State-wise Loan Performance

## Tools Used

* Power BI
* Power Query
* DAX
* Microsoft Excel

## Business Problem

Financial institutions need visibility into loan performance, repayment behavior, and portfolio risk.

This dashboard provides insights into loan quality by comparing funded amounts, repayments, and loan status classifications.

## Key KPIs

* Total Loan Applications
* Total Funded Amount
* Total Amount Received
* Average Interest Rate
* Average DTI Ratio
* Good Loan Percentage
* Bad Loan Percentage

## Key Insights

* Evaluated loan portfolio health through Good and Bad Loan segmentation.
* Compared funded amounts with repayment collections.
* Identified major loan purposes driving application volume.
* Analyzed borrower risk indicators such as DTI and Interest Rate.
